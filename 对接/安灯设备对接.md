## 向安灯请求推送设备报警处理状态(安灯提供)

**接口地址**:

- `http://10.125.129.52:8001/std/api/datasource/alarm/status/equip`
- `https://p9-andon-10-125-129-52.xip.fii-fmc.com/std/api/datasource/alarm/status/equip`

**请求方式**:`POST`

**请求数据类型**:`application/json`

**请求示例**:

```javascript
[
  {
    id: "smt01t.cm|1_alarm_code",
    t: 1763353866285,
    time: "2025-11-21 12:00:00",
    statusCode: "5",
    statusName: "待评价",
  },
];
```

或

```javascript
[
  {
    id: "smt01t.cm|1_status",
    t: 1763353866285,
    time: "2025-11-21 12:00:00",
    statusCode: "5",
    statusName: "待评价",
  },
];
```

**请求参数**:

| 参数名称   | 参数说明                                                                                                                                                                                                                                           | 是否必须 | 数据类型          |
| ---------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | -------- | ----------------- |
| id         | 原异常id                                                                                                                                                                                                                                           | true     | string            |
| t          | 原异常的时间戳                                                                                                                                                                                                                                     | true     | integer(int64)    |
| time       | 处理的时间（格式2025-11-21 12:00:00）                                                                                                                                                                                                              | true     | string(date-time) |
| statusCode | 处理的状态<br /> WAITING_LIST("待派单", 1),<br/> PENDING_ORDERS("待接单", 2),<br/> TO_BE_SIGNED_IN("待签到", 3),<br/> TO_BE_COMPLETED("待完工", 4),<br/> TO_BE_EVALUATED("待评价", 5),<br/> PENDING_CASE("待结案", 6),<br/> FINISHED("已完成", 7); | true     | string            |
| statusName |                                                                                                                                                                                                                                                    | true     | string            |

**响应示例**:

```javascript
{
  "code": "00000",  // 非00000失败
  "message": "success",
  "payload": null
}
```

## 安灯向设备模块请求推送设备报警(设备模块提供)

**接口地址**:`http://10.125.129.46:8000/api/standardization/equipment/maintenance/insertByAndon`

**请求方式**:`POST`

**请求数据类型**:`application/json`

```javascript
{
  "driver": "panasonic_chip_mounter",
  "timestamp": 1763349766948,
  "values": [
    {
      "id": "smt01t.cm|1_alarm_code",
      "t": 1763353866285,
      "dt": "string",
      "v": "03161502"
    },
    {
      "id": "smt01t.cm|1_alarm_msg",
      "t": 1763353866285,
      "dt": "string",
      "v": "Parts recognition error"
    }
  ]
}
```

**请求参数**:

| 参数名称  | 参数说明                   | 是否必须 | 数据类型       |
| --------- | -------------------------- | -------- | -------------- |
| driver    | 设备名称                   | true     | string         |
| timestamp | 时间戳                     | true     | integer(int64) |
| values    |                            | true     | array          |
| &emsp;id  | 设备报警代码id或报警消息id | true     | string         |
| &emsp;t   | 时间戳                     | true     | integer        |
| &emsp;dt  | v的值类型                  | true     | string         |
| &emsp;v   | 值                         | true     | string         |

**响应示例**:

```javascript
{
    "code": "200",
    "msg": null,
    "data": "OK",
    "success": true
}
```
